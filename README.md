# Личная кинотека

Автор: Иван Иванов

Описание:
Приложение для хранения информации о фильмах. Позволяет добавлять фильмы, фильтровать их по жанру и году выпуска, сохранять данные между запусками с помощью JSON.

Использование:
- Введите название, жанр, год и рейтинг фильма.
- Нажмите «Добавить фильм» для добавления.
- Используйте выпадающие списки для фильтрации по жанру и году.
- Все данные сохраняются автоматически при добавлении.

Пример теста:
- Введите фильм: "Интерстеллар", жанр: "Фантастика", год: 2014, рейтинг: 9.0
- Нажмите «Добавить фильм».
- Выберите жанр «Фантастика» или год «2014» для фильтрации.
